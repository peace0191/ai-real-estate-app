# Global Antigravity Rules

## 프로젝트 설명

이 프로젝트는 Python + Streamlit 기반 부동산 AI 플랫폼이다.

## 주요 기능

- 매물 등록
- AI 저평가 분석
- AI 추천 매물
- 고객 매칭
- 숏츠 광고 관리
- 고객 관리

## 수정 규칙

- app.py 핵심 로직은 함부로 삭제하지 않는다.
- 매물 데이터 구조는 유지한다.

## 기술 스택

- Python
- Streamlit
- Pandas
- SQLite 또는 CSV 데이터

## UI 규칙

- 모든 메뉴는 왼쪽 사이드바에 배치
- 한 화면에 하나의 주요 기능 표시

## AI 기능

- 매물 가격 분석
- 저평가 여부 판단
- 투자 추천 매물 생성

## 금지 규칙

- 매물 데이터 삭제 금지
- 고객 정보 무단 수정 금지
